The first leetcode problem was quite simple. There is a list of interaction pairs, that follow a format:

["User A", "User B", "Likes"] -> User A likes User B

Return the count of mutual interaction pairs. That is, if A interacts with B AND B interacts with A, then A-B is a pair. Note that A-B and B-A are the same pair and are not counted as two distinct pairs.

Thoughts: Note that the action itself doesn't really matter. We just want to count A->B and B->A.

Solution: Let there be a s = set() and a ret = 0. Iterate through all interactions. For every A interact B, if (A, B) not in s, s.add((A,B)). Check if (B, A) is in s. If so, increment ret by 1. We should be able to count all unique mutual interaction pairs. Return ret, the count of such pairs.
